### Installation

Step-by-step instructions. Example:

```bash
# clone the repo
git clone https://github.com/YourUsername/RecollectAi_SupaStrikas.git  
cd RecollectAi_SupaStrikas

# install backend dependencies
cd server
npm install

# install frontend dependencies
cd ../client
npm install

# in server/.env
PORT=5000
DB_URI=your_database_connection_string
AI_API_KEY=your_ai_key

# start the backend
cd server
npm run start

# in another terminal, start front-end
cd client
npm run dev
```
